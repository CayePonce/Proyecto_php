Hola, bienvenido a esta prueba técnica. 
Instrucciones de uso:
En localhost:8000/newMovie tienes un pequeño formulario hecho con twig para introducir el ID de la película, al darle a submit te devolverá un json donde te dará un mensaje diciendo que tienes la película guardada en la base de datos.
En localhost:8000/movies tienes un listado de todas las peliculas que tiene la base de datos

En caso de que el .env venga vacío, este es mi enlace a la base de datos : DATABASE_URL="mysql://root:@127.0.0.1:3306/movies_db?serverVersion=10.11.2-MariaDB&charset=utf8mb4"
